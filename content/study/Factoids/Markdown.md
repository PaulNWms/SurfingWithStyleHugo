### Markdown reminders

| HTML | markdown |
| --- | ---|
| `<br/>` | end a line with 2 spaces |
| `<hr/>` | `---` or `***` (thick) |
| `<a href="link">text</a>` | `[text](link)` |
| `<img src="path.jpg" alt="alt text" />` | `![alt text](path.jpg)` |
