You can set up a prebuilt theme by using

- Hugo modules
- Git submodules
- downloading and copying to the themes folder

Then add a line to `hugo.yaml` (or `config.yaml`)

```yaml
...
theme: <name>
```

Many themes have an `exampleSite` folder.
